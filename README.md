# Aquaglass Catálogo y Presupuesto

Web visual para navegar productos reales del catálogo Aquaglass, armar una selección comercial y generar un PDF de presupuesto.

## Abrir

Opción simple:

1. Abrir `index.html` en el navegador.

Opción con servidor local:

1. Abrir una terminal en esta carpeta.
2. Ejecutar `node serve.js`.
3. Entrar a `http://127.0.0.1:8080/`.

## Datos editables

- Catálogo visual: `data/catalog.json`.
- Respaldo para abrir directo por archivo: `data/catalog.js`.
- Fotos reales copiadas: `assets/catalogo-real/`.
- Logo y recursos base: `assets/`.

Los precios son ficticios, en pesos argentinos, dentro de un rango de referencia de $1 a $1.000.
